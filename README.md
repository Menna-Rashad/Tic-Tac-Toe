# 🎮 React Tic-Tac-Toe

A modern, responsive Tic-Tac-Toe game built as a hands-on project to master React state management and UI styling. 

I developed this project following the official React documentation, but I took the UI a step further by implementing a custom, pixel-perfect design using **Tailwind CSS**.

## ✨ Features
- **Modern UI:** Custom color palette and clean interface.
- **Responsive Design:** Fully responsive layout optimized for all screen sizes (Mobile to Desktop) using Tailwind's utility classes.
- **State Management:** (Coming soon) Handling game logic, player turns, and winner calculation using React Hooks (`useState`).

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- HTML5 / CSS3
